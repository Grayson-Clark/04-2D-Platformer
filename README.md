# 2D-Platformer-Character

A 2D Platformer game.
Project 04 for C220, March 23, 2021.

## Implementation
Built using Godot 3.2.3

- Walljumps
- Saving
- Loading
- Music
- Player health
- Score
- Coins
- Mines
- Turrets
- Introductory/Ending "hub world"
- 2 Levels
